>[!propriete] Propriété :
>On peut additionner (ou soustraire) deux fractions de même dénominateur. Pour cela, on additionne (ou soustrait) les numérateur et on conserve le dénominateur commun. 

>[!exemple] Exemple :
>- $\frac{5}{7}+\frac{3}{7}=\frac{5+3}{7}=\frac{8}{7}$
>- $\frac{7}{11}-\frac{3}{11}=\frac{7-3}{11}=\frac{4}{11}$

>[!methode] Méthode :
>Si les dénominateurs sont différents, on transforme une fraction (ou les deux), avec les [[Modules/Modules 5e/Fractions égales\|fractions égales]] pour obtenir le même dénominateur. 

>[!exemple] Exemple :
>$$\begin{aligned}A&=\frac{3}{5}+\frac{7}{15}\\&=\frac{3 \times 3}{5\times 3}+\frac{7}{15}\\&=\frac{9}{15}+\frac{7}{15}\\&=\frac{16}{15}\end{aligned}$$
>$$\begin{aligned}B&=\frac{2}{3}-\frac{1}{2}\\&=\frac{2\times 2}{3 \times 2}-\frac{1 \times 3}{2 \times 3}\\&=\frac{4}{6}-\frac{3}{6}\\&=\frac{1}{6}\end{aligned}$$
