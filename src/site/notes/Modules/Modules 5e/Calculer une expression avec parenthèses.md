# Calculer une expression avec parenthèses
>[!propriete] Propriété (admise) :
>On effectue en premier les calculs contenus dans les parenthèses les plus intérieur. 

>[!methode] Méthode : Calculer une expression
>Pour calculer une expression :
>1. On repère le calcul prioritaire ;
>2. On écrit son résultat juste en dessous ;
>3. On recopie le reste de l’expression sans changer l’ordre.

>[!exemple]
>$$\begin{aligned}A&=3\times (5+\underline{(6-2)})\\A&=3\times \underline{(5+4)}\\A&=\underline{3\times 9}\\A&=27\end{aligned}$$
