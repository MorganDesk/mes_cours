>[!propriete] Propriété (admise) :
>Entre deux nombres relatifs, le plus grand est celui qui se trouve le plus à droite sur la droite graduée. Donc :
>- **Entre deux nombres négatifs**, le plus grand est celui qui a la **plus petite** distance à zéro.
>- **Entre deux nombres positifs**, le plus grand est celui qui a la **plus grande** distance à zéro.
>- **Entre un nombre négatif et un nombre positif**, le plus grand est toujours le plus grand.

>[!exemple] Exemples :
>- $(+2) < (+7)$
>- $(-3) < (+5)$
>- $(-9) < (-4)$
