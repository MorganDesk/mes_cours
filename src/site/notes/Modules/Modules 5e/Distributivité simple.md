>[!propriete] Propriété : Distributivité simple
>Pour tous nombres *k*, *a* et *b* :
>$$k(a+b)=ka+kb$$
>et 
>$$k(a-b)=ka-kb$$
{ #76c514}

>[!remarque] Rappel :
>La distributivité simple peut être utile pour le [[Modules/Modules 5e/Distributivité simple et calcul réfléchi\|calcul réfléchi]].
## Développer
>[!definition] Définition : 
>**Développer** c’est transformer un produit en une somme ou une différence.
>Pour cela, on utilise les formules de distributivité.

>[!exemple] Exemple :
>Développer $A=3(2x+1)$ :
>$$\begin{aligned}A&=3 \times 2x + 3 \times 1\\&=6x+3\end{aligned}$$

## Factoriser
>[!definition] Définition :
>**Factoriser** c’est transformer une somme ou une différence en un produit. 
>C’est l’inverse du développement. On cherche un **facteur commun**.

>[!exemple] Exemple :
>- Factoriser $B=16x+8$. Le facteur commun est 4.
>$$\begin{aligned}B&=4\times 4x + 4\times 2\\&=4(4x+2)\end{aligned}$$
>- Factoriser $C=2x^2-5x$. Le facteur commun est $x$ (*Rappel : $x\times x=x^2$*) 
>$$\begin{aligned}C&=x\times 2x -x \times 5\\&=x(2x-5)\end{aligned}$$
