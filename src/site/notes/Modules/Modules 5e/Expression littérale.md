# Expression littérale
>[!definition] Définition :
>Une **expression littérale** est une expression mathématique contenant une ou plusieurs lettres qui désignent des nombres (variables ou inconnues).

## Modélisation de propriétés géométriques
On peut utiliser une expression littérale pour écrire des formules générales :
- **Périmètre d’un carré** de côté *c* :
$$P=4 \times c$$
- **Aire d’un disque** de rayon *r* :
$$A=\pi \times r \times r$$
## Modélisation d’un programme de calcul
Considérons le programme de calcul suivant :
1. Choisir un nombre
2. Multiplier par 3
3. Ajouter 2
Si on appelle *x* le nombre choisi au départ (la variable) :
$$x \longrightarrow x \times 3 \longrightarrow x \times 3 + 5$$
