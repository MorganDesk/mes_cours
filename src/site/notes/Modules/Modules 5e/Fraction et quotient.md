>[!definition] Définition :
>Soient deux nombres *n* et *d* ($d \neq 0$). Le quotient de *n* par *d* est le nombre qui, multiplié par *d*, donne *n*. 
>- *n* est le **numérateur**.
>- *d* est le **dénominateur**.
>
>On peut écrire : 
>$$\frac{n}{d}\times d=n \text{ et } n \div d = \frac{n}{d}$$

>[!exemple] Exemple :
>- Le nombre $\frac{5}{4}$ est le quotient de 5 par 4. C’est le nombre qui, multiplié par 4 donne 5 :
>$$\frac{5}{4}\times 4=5$$
>Sa valeur décimale est $5 \div 4=1,25$.
>- Le nombre $\frac{2}{3}$ est le quotient de 2 par 3. C’est le nombre qui, multiplié par 3 donne 2 :
>$$\frac{2}{3}\times 3=2$$
>Sa valeur décimale est $2 \div 2 \approx 0,66$… qui n’est pas un [[Nombre décimal\|nombre décimal]]. L’écriture $\frac{2}{3}$ est sa valeur exacte.

>[!definition] Définition :
>Une **fraction** est une écriture fractionnaire dont le numérateur et le dénominateur sont des [[Modules/Modules généraux/Nombre entier\|nombres entiers]].
