>[!definition] Définition :
>Un [[Modules/Modules généraux/Nombre entier naturel\|nombre entier naturel]] est **premier** s'il n'admet que deux [[Modules/Modules 6e/Multiple et diviseur#^461d4a\|diviseurs]] distincts : 1 et lui-même.
{ #2c3c7f}


>[!exemple] Exemple :
>Les nombres premiers inférieurs à 30 sont : 2 ; 3 ; 5 ; 7 ; 11 ; 13 ; 17 ; 19 ; 23 et 29.

>[!remarque] Remarque :
>Il existe une infinité de nombre premiers.

>[!exercice] Exercice :
>Justifier que les nombres suivant ne sont pas des nombres premiers : $$573 ; 1784 ; 1065$$

>[!exercice]- Correction :
>- 573 : $5+7+3=15$ ; 15 est divisible par 3 dont 573 aussi. 573 n’est pas premier.
>- 1784 : ce nombre est pair donc divisible par 2. 1784 n’est pas premier.
>- 1065 : son chiffre des unités est 5, il est divisible par 5. 1065 n’est pas premier.

>[!exercice] Exercice : (extrait du DNB Métropole Septembre 2017)
>**Affirmation** : Pour tous les nombres entiers *n* compris entre 2 (inclus) et 9 (inclus), l’entier $2^n -1$ est un entier premier.
>Cette affirmation est-elle vraie ou fausse ?

>[!exercice]- Correction :
>Pour $n=4$ on a $2^4 -1=16-1=15$.
>Or 15 n’est pas un entier premier puisque $15=3 \times 5$.
>L’affirmation est fausse.
