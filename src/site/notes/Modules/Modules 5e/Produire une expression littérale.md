>[!methode] Méthode : Produire une expression littérale
>- On choisit une lettre pour représenter l’information recherchée.
>- On traduit les mots-clés en opérations :
>	- Somme / Ajouter / Augmenter / … → $+$
>	- Différence / Soustraire / Diminuer / … → $-$
>	- Produit / Multiplier / Doubler ($\times 2$) / Tripler ($\times 3$) / … → $\times$ 
>	- Partager / Diviser / Moitié ($\div 2$) / … → $\div$
>- On fait un test numérique pour répondre au problème.
>- On reprend le test précédent et on remplace le nombre choisit « au hasard » par la lettre choisit au début. 

>[!exemple] Exemple :
>Dans un cinéma on trouve deux formules :
>- Formule A : Une place coûte 8€.
>- Formule B : On achète un abonnement à 15€ qui permet de payer chaque ticket d’entrée 5€.
>Pour chaque formule, écrire l’expression qui permet de calculer le prix payé en fonction du nombre de tickets achetés.
>
>On appelle $n$ le nombre de tickets achetés.
>
>Pour la formule A, si on achète 5 places, on paiera $5 \times 8 = 40\texteuro$.
>L’expression littérale de la formule A est donc $n \times 8=8n$.
>
>Pour la formule B, si on achète 4 place, on paiera $15 + 4 \times 5=15+20=35\texteuro$.
>L’expression littérale de la formule B est donc : $15 + n \times 5=15+5n$.
