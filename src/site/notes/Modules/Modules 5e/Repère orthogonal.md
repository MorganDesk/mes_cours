>[!definition] Définitions :
>Un **repère orthogonal** du plan est constitué de deux [[Modules/Modules 5e/Repérage sur une droite graduée#^94e691\|droites graduées]] perpendiculaires et de même origine :
>- la droite horizontale, orientée vers la droite est l’**axe des abscisses** ;
>- la droite verticale, orientée vers le haut, est l’**axe des ordonnées**.
>
>L’intersection des deux axes est l’**origine du repère**.
>
>Chaque point du plan est repéré par son **abscisse** et son **ordonnée**. Ces deux nombres forment les **coordonnées** du point.
>
>Si un point $M$ a pour abscisse $a$ et pour ordonnée $b$, on notera $M(a;b)$.

>[!exemple] Exemple :
 ![repère orthogonal.png](/img/user/Images/rep%C3%A8re%20orthogonal.png)
>- Dans le repère orthogonal, l’abscisse du point A est 3, son ordonnée est -1. On note $A(3;-1)$.
>- De même pour B, on note $B(-1;2)$.
>- Le point C est sur l’axe des abscisses, son ordonnée est nulle : $C(2;0)$.
>- Le point D est sur l’axe des ordonnées, sont abscisse est nulle : $D(0;-3)$.

>[!remarque] Remarques :
>- L’abscisse d’un point est nulle s’il se trouve sur l’axe des ordonnées.
>- L’ordonnée d’un point est nulle s’il se trouve sur l’axe des abscisses.
