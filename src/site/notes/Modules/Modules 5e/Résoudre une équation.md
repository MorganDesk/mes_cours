>[!definition] Définition :
>Une **équation** est une égalité d’[[Modules/Modules 5e/Expression littérale\|expressions littérales]] qui peut être vraie ou fausse. Les lettres dans ces expressions sont les **inconnues**.
>**Résoudre** une équation, c’est trouver toutes les valeurs des inconnues qui rendent l’égalité vraie. Ces valeurs sont les **solutions** de l’équation.

>[!remarque] Remarques :
>- On peut utiliser n’importe quelle lettre pour désigner l’inconnue dans une équation.
>- Une équation peut avoir 0 ; 1 ou plusieurs solutions.
