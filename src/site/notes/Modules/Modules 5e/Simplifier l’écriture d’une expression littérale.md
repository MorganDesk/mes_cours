>[!propriete] Règle de simplification d’écriture :
>On peut supprimer le signe de multiplication $\times$ s’il est suivi d’une lettre ou d’une parenthèse.

>[!exemple] Exemple :
>- $3 \times x=3x$
>- $a \times b =ab$
>- $2\times (x+3)=2(x+3)$

>[!definition] Notations :
>- $c\times c$ se note $c^2$ et se lit « *c* au carré »
>- $p \times p \times p$ se note $p^3$ et se lit « p au cube »

>[!remarque] A connaître par coeur :

| $0^2=0$    | $1^2=1$     | $2^2=4$    |
| ---------- | ----------- | ---------- |
| $3^2=9$    | $4^2=16$    | $5^2=25$   |
| $6^2=36$   | $7^2=49$    | $8^2=64$   |
| $9^2=81$   | $10^2=100$  | $11^2=121$ |
| $12^2=144$ | $10^3=1000$ |            |
