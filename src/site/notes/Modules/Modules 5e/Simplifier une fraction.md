>[!definition] Définition :
>**Simplifier** une fraction revient à trouver une [[Modules/Modules 5e/Fractions égales\|fraction égale]] dont le numérateur et le dénominateur sont plus petits.

>[!exemple] Exemple :
>$$\frac{14}{21}=\frac{14\div 7}{21 \div 7}=\frac{2}{3}$$

>[!definition] Définition :
>Une fraction est dite **irréductible** quand elle ne peut pas être simplifiée.

>[!exemple] Exemple :
>Simplifier la fraction $\frac{30}{45}$ :
>$$\frac{30}{45}=\frac{30 \div 5}{45\div 5}=\frac{6}{9}=\frac{6 \div 3}{9 \div 3}=\frac{2}{3}$$
>La fraction $\frac{2}{3}$ ne peut pas être simplifier, elle est irréductible.

>[!remarque] Remarque :
>Les [[Modules/Modules 6e/Critères de divisibilité\|critères de divisibilité]] sont très utiles pour simplifier les fractions.
