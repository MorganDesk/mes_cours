# Vocabulaire des opérations
>[!definition] Définition :
>- Le résultat d’une addition est une **somme**. Les nombres dans l’additions sont les **termes**.
>- Le résultat d’une soustraction est une **différence**. Les nombres dans la soustraction sont les **termes**.
>- Le résultat d’une multiplication est un **produit**. Les nombres dans la multiplication sont les **facteurs**.
>- Le résultat d’une division est un **quotient**.

>[!exemple] Exemple :
>- $A=2+3\times 5$ est une somme car la dernière opération effectuée est une addition.
>- $B=8\times (6-2)$ est un produit car la dernière opération effectuée est une multiplication.
